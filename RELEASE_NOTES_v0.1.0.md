# rwrsb_gui v0.1.0

## 中文

### 亮点

- 支持读取 `.vox` 和当前项目使用的 XML
- 支持 `particle` / `stick` 骨架结构增删改
- 支持体素绑定刷选、框选、解绑和按骨段重绑
- 支持骨架预设的保存、加载、覆盖和删除
- 支持在视口中直接拖拽粒子点
- 支持网格显示、主次网格、三平面网格和网格吸附
- 支持中英双语界面
- 支持 UI 缩放
- 支持相机 Y 轴反转

### 适用环境

- Windows
- Python 源码运行，或使用发布包中的 `rwrsb_gui.exe`
- 需要可用的 OpenGL 驱动

### 已知说明

- 当前发布包为目录版，不是单文件版
- 着色器和预设资源已随发布包一起打入
- 不同机器上的显卡驱动兼容性仍建议实机测试

## English

### Highlights

- Load `.vox` files and project-compatible XML
- Full `particle` / `stick` skeleton CRUD
- Voxel brush/select/unbind/rebind workflow
- Skeleton preset save/load/overwrite/delete
- Direct particle dragging in the viewport
- Grid display, major/minor grid, 3-plane grid, and snapping
- Chinese / English UI
- UI scaling
- Invert-Y camera option

### Environment

- Windows
- Run from source with Python or from the packaged `rwrsb_gui.exe`
- Requires working OpenGL drivers

### Notes

- This release ships as a directory-based package, not a single-file exe
- Shader and preset resources are bundled with the build
- Real-machine GPU/driver validation is still recommended
